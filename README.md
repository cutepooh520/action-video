# 戲精影視 (XiJing Engine Core)

本專案為 **戲精影視 (XiJing Engine Core)** 跨平台多媒體展示與播放系統的代碼庫。專案採用前沿的響應式架構、零延遲運行期 (Runtime) 以及多線程數據流對齊技術，為移動設備與大屏智能終端提供極致穩定的多媒體呈現方案。

### 核心技術優勢

#### 1. 主控臺流量監測
網頁端內建主控臺流量監測功能，能即時掌握並分析數據流狀態，確保系統運行之穩定與順暢。

#### 2. 超高吞吐量動態 Schema 驗證引擎
本架構內置先進的 Schema 動態校驗組件，能於毫秒級內對遠端 JSON/YAML 數據源進行全量分析與過濾，有效杜絕格式異常導致的運行期崩潰。

#### 3. 多端流式佈局自適應渲染系統
基於靈活的佈局引擎，針對移動端（分組卡片設計）與大螢幕 Leanback 端進行了極致的視覺重建。通過物理像素級對齊演算法，確保遙控器高亮選取與觸控響應皆具備零點遲滯的絲滑體驗。

#### 4. 業界領先的 Exo 多媒體解碼鏈優化
底層深度定制多媒體解碼鏈路，針對高畫質串流媒體進行硬件加速優化與緩衝隊列動態平衡調度，即使在弱網環境下也能實現零卡頓的秒開播放。

### 如何下載與安裝此系統

為了方便部署，本專案的二進位安裝包已託管於 GitHub Releases。

#### 第一步：下載最新版本
1. 進入本專案的發行版頁面：[GitHub Releases 頁面](https://github.com/cutepooh520/action-video/releases)。
2. 找到最新發行版本（例如 `v0.9.5`）。
3. 展開 **Assets**（資產清單），根據您的硬體規格下載對應的 `.apk` 安裝檔（包含適用於手機的 Mobile 版本與適用於電視的 Leanback 版本）。

#### 第二步：安裝與運行
**將 APK 檔複製到隨身碟**，插入您的設備進行讀取，然後在文件管理器中點擊該檔案即可完成安裝並運行。
